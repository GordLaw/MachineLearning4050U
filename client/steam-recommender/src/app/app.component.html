Find Your Next Favorite Game

Find your next game with our AI-powered recommender system to analyzes your tastes and playstyle to deliver smart, personalized game suggestions.

About This Project

This project is made by Daniel Bryon, Beatriz Provido, and Gordon Law for our final project of machine learning 4050U. Compiling what we've learnt through the course and the research we have done to create a model which uses a graph neural network (GNN) to generate personalized game recommendations from a given input through link prediction. Utilizing libraries such as Pytorch, Geometric, and Pandas to capture the patterns and deeper relationships between users and the games they play.
